# 🇰🇪 Sovereign-Ledger | Nairobi Node v1.0

A professional-grade, event-sourced financial engine designed to manage capital with absolute mathematical certainty. This is a **Hybrid Sovereign System** where Ruby manages the "Truth" and Python manages the "Intelligence."

## 🏗️ Technical Architecture
- **Immutable Core:** Ruby on Rails utilizing a "Fact-Based" Ledger (Events, not just Rows).
- **Analytical Layer:** Python AI Agent (`sovereign_analyst.py`) for burn-rate and risk assessment.
- **Interface:** High-performance "Luxury" Dark Theme (Tailwind CSS / DaisyUI).
- **Locality:** Optimized for KES (Kenyan Shilling) and EAT (East Africa Time).

## ⚡ Quick Start
1. **Launch the Ledger:** `bin/rails server`
2. **Initialize AI Analyst:** `python3 sovereign_analyst.py`
3. **Access Dashboard:** `http://localhost:3000/wallets/1`

## 🛡️ Security & Integrity
Every transaction is stored as a unique "Fact" with a cryptographic Hash, ensuring that the audit trail can never be tampered with or deleted without detection.
